polr-docker-compose
===================

Very simple to use

```
$ export POLR_SERVER_NAME="polr.example.com p.ex"
$ export POLR_DB_PASSWD="example"
$ export POLR_DB="polr"
$ docker-compose up -d
```

Next go to your host setted in ENV variable POLR_SERVER_NAME and you must see the setup page of polr project.
